# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [2.2.0] - 2026-03-29

### Added

- **Move Tab to Start** and **Move Tab to End** actions (assign shortcuts via Settings > Keymap).
- Configurable wrap-around behavior for Move Tab Left/Right (Settings > Editor > Move Tab). Wrap-around is enabled by default.

### Changed

- Minimum supported IntelliJ version updated to 2025.3. No maximum version is set; the plugin is intended to remain compatible with all future versions.

## [2.1.1] - 2023-10-22

### Fixed

- Added missing changelogs (again).

## [2.1.0] - 2023-10-22

### Changed

- Improved performance when moving tabs.
- Addressed deprecation warnings for IntelliJ 2023.2.

## [2.0.0] - 2022-10-11

### Changed

- Added support for IntelliJ 2022.3.
- Removed deprecated method usages.
